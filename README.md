# Gismeteo-PHP

Library that implements weather service "Gismeteo" API in PHP

You need get GISMETEO Token. For request token send email to: `b2b@gismeteo.ru`

## How to use:

Set
`$_SERVER['GISMETEO_TOKEN']`

See implemetations:

```
GismeteoInterface
```

`use SakuraSamurai\Gismeteo\Gismeteo;`

`new Gismeteo();`

### Good luck ;3
